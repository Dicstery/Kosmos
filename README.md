# Kosmos

Минималистичный 3D-движок реального звёздного неба в браузере.  
Никаких процедурных генераций — только астрономические данные из реальных каталогов.

---

## Возможности

- **Солнечная система** — Солнце, планеты, спутники, кольца, атмосфера
- **Реальные звёзды** — каталог AT-HYG (300 000+ звёзд) с точными координатами
- **Свободный полёт** — экспоненциальное ускорение камеры как в Celestia
- **День/ночь на Земле** — кастомный шейдер
- **Облака на газовых гигантах** — процедурный шум
- **HDR и bloom** — постобработка для ярких объектов
- **Локальный сервер** — запуск одной командой

---


Управление

· Мышь: вращение камеры, зум
· Клик по планете: приближение и информация
· Кнопка ×: возврат камеры
· Кнопки вверху: яркость, орбиты, GUI
